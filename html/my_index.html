인용문 관련 태그

blockquote :

안쪽의 텍스트가 긴 인용문임을 나타냄. 들여쓰기를 한 것으로 그려짐.
cite 속성으로 출처 표시

q : 짧은 인용문에 사용
cite 속성으로 출처 표시

mark : 인용문 중 하이라이트 또는 사용자 행동과 연관된 곳 표시(예: 페이지 검색)

abbr : FIFO 준말/머릿글자 표시. title 속성으로 원래 형태 표시하며 글자 위에 hover시 확인 가능

ul : 순서가 없는 목록
ol : 순서가 있는 목록. type, start 속성 사용 가능.
li : ul, ol 태그의 1촌 자식으로 이 태그만 가능

용어와 정의 나열하는 태그
dl : dl 태그 안에서 사용
dt : 단어
dd : 정의

table: 테이블
caption: 표 설명 또는 제목 (선택)
tr : 테이블 행
td : 테이블 데이터(셀)

표를 설명하는 곳
1행 1열 1행 2열 1행 3열
2행 1열 2행 2열 2행 3열
3행 1열 3행 2열 3행 3열

thead: tbody 앞에 와야 함
tbody: 본 내용을 담음
tfoot: tbody 뒤에 와야 함
th: scope 속성으로 row, col 중 선택

  • _self : 현재 창에서 이동(기본)

  • mailto:email@example.com 하면 이메일로 이동

입력값 받는 태그

form :
정보를 제출하기 위한 태그들을 묶음. input, label, button이 안에 있음.
autocomplete 속성: 자동완성 여부(생략시 on)

input :
사용자로부터 입력을 받는 요소
input의 타입과 속성
name: 정보들을 서버로 보낼 때 사용. 서버에서 각 항목을 받을 때 무엇에 대한 값인지 구분하는 이름으로 작용.
value: 값이 지정됨
autofocus: 자동 포커스
readonly: 읽기만 가능, 전송됨
required: 필수 입력값
disabled: 입력 불가, 전송 안됨
type속성을 통해 다양화








checkbox : value로 값을 넘겨줌


radio : name으로 묶으면 1개만 선택 가능

label: 라벨의 for속성값과 input의 id를 연결하면 라벨을 클릭해도 인풋이 선택됨. (같은 폼 안에서)

button:
type submit : 제출, type reset : 초기화, type button : 기본 동작 없음

fieldset:
폼 태그 내 입력요소와 라벨들을 그룹화
disabled 속성 : 포함된 입력요소 비활성화

legend:
필드셋 요소의 제목 또는 설명

회원가입

select, option: 옵션중 선택하기

optgruop: 옵션을 그룹으로 묶음

datalist: 입력값 추천
